### ÂżPor quĂ© no se debe usar `allow_origins=["*"]` junto con `allow_credentials=True`?

Si se usara `"*"` como origen permitido junto con `allow_credentials=True`, cualquier sitio web â€”incluso uno maliciosoâ€” podrĂ­a enviar peticiones autenticadas en nombre del usuario que tenga la pestaĂ±a abierta, ya que el navegador adjuntarĂ­a las credenciales sin restricciĂłn de origen.
